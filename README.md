# Basis Task Backend

## Libraries used
- Node.js
- Express.js
- body-parser
- cors
- morgan
- mongoose
- jsonwebtoken
- nodemailer

## Installation
    git clone https://github.com/sauravpratihar/basis-task-be.git
    cd basis-task-be
    npm i

## Run
To start the node backend server run following command on terminal

    npm start
